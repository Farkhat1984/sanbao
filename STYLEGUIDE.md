**Стиль:** Soft Corporate Minimalism

---

| `--bg-surface-alt` | `#F3F5F9` | `#1E2433` | Сайдбар, поля ввода |

**Правило:** фон никогда чисто белый/чёрный — всегда с лёгким голубым оттенком.

## Типографика

- **Шрифт:** Inter (Geist Sans) — весь интерфейс
- **Моноширинный:** JetBrains Mono (Geist Mono) — код, юр. коды статей
- **Base size:** 14px, line-height 1.6
- **Заголовки:** 600–700 weight
- **Основной текст:** 400–500 weight

## Ключевые паттерны

### Кнопки
- Primary: залитая `--accent`, белый текст
- Secondary: `--surface-alt` фон + бордер
- Ghost: прозрачная, текст меняет цвет при hover
- Gradient: `from-accent to-legal-ref` — для главных CTA
- Все кнопки: `active:scale-[0.98]` — микро-отклик

### Поле ввода чата
- Не приклеено к низу — «парит» с отступами
- Радиус 32px, белый фон с бордером
- При фокусе: градиентная обводка (indigo→purple) через `mask-composite`
- Кнопка отправки: круглая с градиентом, появляется когда есть текст

### Сайдбар
- Glassmorphism: `backdrop-filter: blur(16px)`, полупрозрачный фон
- Сворачивается до 64px с spring-анимацией
- Группировка чатов: Сегодня / Вчера / Эта неделя / Ранее
- Лого: квадрат с градиентом + иконка весов

### Сообщения
- User: залитый `--accent`, белый текст, скруглён сверху-справа `rounded-tr-md`
- Assistant: `--surface-alt` фон + бордер, скруглён сверху-слева `rounded-tl-md`
- Появление: `opacity 0→1, translateY 8px→0` за 200ms

### Юридические ссылки
- Фиолетовый бейдж (`--legal-ref-bg` + `--legal-ref`)
- Иконка книги + название статьи + индикатор актуальности (зелёный ✓ / жёлтый ⚠)
- Клик → попап с текстом статьи

### Артефакт-панель
- Выезжает справа, 480px, spring-анимация
- Три таба: Просмотр / Редактор / Исходник — с `layoutId` анимацией
- Тулбар: копировать, скачать, печать

### Анимации
- Spring: `damping: 25, stiffness: 300` — для layout-переходов
- Ease: `cubic-bezier(0.4, 0, 0.2, 1)` — для hover/fade
- Stagger: 60ms между элементами в списках
- Индикатор «думает»: покачивание весов ±8° за 2s

## Иконки

Lucide React. Размер 16–18px (`h-4 w-4`). Цвет `--text-muted`, при hover `--text-primary`.

---

## Отступы

- Между секциями: `24–32px`
- Padding карточек: `16–24px`
- Gap между элементами: `8–12px`
- Padding сообщений: `12px 16px`

---

## Правила

1. Никогда чисто белый/чёрный фон — всегда с оттенком
2. Бордеры вместо теней для разделения — тени только для «парящих» элементов
3. Градиент `accent→legal-ref` — только для 1–2 главных CTA на экране
4. Анимации обязательны, но незаметны — spring для layout, ease для цвета
5. Все интерактивные элементы: hover-состояние + `cursor-pointer`
6. Текст ошибок — `--error`, успех — `--success`, без лишних иконок
